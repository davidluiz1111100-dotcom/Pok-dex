# Pokédex

Uma Pokédex feita com HTML, CSS e JavaScript.

## O que ela faz

- Carrega Pokémon usando a PokeAPI.
- Mostra nome, número, imagem, tipos e estatísticas.
- Permite pesquisar por nome ou número.
- Permite filtrar por tipo.
- Funciona direto no navegador.

## Tecnologias

- HTML
- CSS
- JavaScript
- PokeAPI
